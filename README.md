# OsintForge

**Next-generation Open Source Intelligence (OSINT) platform.**

OsintForge is a modern intelligence platform designed to simplify the collection, aggregation, and analysis of publicly available information. By combining data from multiple OSINT sources into a unified interface, OsintForge helps investigators, researchers, journalists, cybersecurity professionals, and businesses gather actionable intelligence more efficiently.

> **Status:** 🚧 Under Active Development


>  https://osintforge.dev

---

## Features

* 🔎 Unified OSINT search interface
* 🌐 Multi-source intelligence aggregation
* 📊 Structured, easy-to-read reports
* ⚡ Fast and responsive web interface
* 📁 Exportable investigation results
* 🔒 Privacy-focused design
* 📱 Responsive desktop and mobile experience

## Development

Clone the repository:

```bash
git clone https://github.com/Recasted/osint-forge.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
